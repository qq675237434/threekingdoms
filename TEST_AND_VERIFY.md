## 🐛 问题排查

### 问题 1: 卡在加载界面
**检查**:
1. 按 F12 打开控制台
2. 查看是否有红色错误
3. 刷新页面重试

**解决**:
```javascript
// 在控制台输入
window.game.init()
```

### 问题 2: 开始界面没反应
**检查**:
1. 确保点击了游戏画面（获取焦点）
2. 按空格键

**解决**:
```javascript
// 在控制台输入
window.game.startGameDirectly()
```

### 问题 3: 角色选择后没进入游戏
**检查**:
1. 确认按了 J 键确认
2. 等待 0.5 秒确认时间

**解决**:
```javascript
// 在控制台输入
startGame()
```

### 问题 4: 黑屏或白屏
**检查**:
1. 浏览器是否支持 Canvas
2. 控制台是否有 WebGL 错误

**解决**:
- 使用 Chrome/Firefox/Edge 等现代浏览器
- 更新显卡驱动
