## 🎮 游戏流程

1. **启动游戏** → 显示开始界面
2. **按任意键** → 可直接开始（默认关羽）或选择"选择武将"
3. **角色选择** → 使用 ←→ 选择，按 J 确认
4. **第 1 关** → 涿郡之战，3 波敌人 + 张飞 BOSS
5. **第 2 关** → 虎牢关，3 波敌人 + 吕布 BOSS
6. **第 3 关** → 赤壁之战，3 波敌人 + 曹操 BOSS
7. **通关** → 显示胜利画面

## 🎯 游戏平衡性

### 难度曲线
- 第 1 关：教学性质，敌人 HP 50，攻击 8
- 第 2 关：难度提升，敌人 HP 60-80，攻击 10-15
- 第 3 关：最高难度，敌人 HP 70-90，攻击 12-20

### BOSS 强度
- 张飞：HP 300，攻击 25（教学 BOSS）
- 吕布：HP 500，攻击 40（高难度）
- 曹操：HP 600，攻击 35（综合难度）

### 道具掉落
- 金创药：10%（普通），30%（BOSS）
- 魔力水：5%（普通），20%（BOSS）
- 战神符：1%（精英），10%（BOSS）
- 天书：0.5%（精英），5%（BOSS）
- 太平要术：100%（最终 BOSS）
