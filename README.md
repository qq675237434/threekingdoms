# 三国战记 - Three Kingdoms Battle

🎮 基于 HTML5 Canvas 的经典街机风格横版动作游戏

## 🎯 项目简介

复刻经典街机游戏《三国战记》的核心玩法，使用纯原生 JavaScript 开发，无需任何外部依赖。包含完整的关卡系统、BOSS 战、道具系统和连击系统。

## 🚀 快速开始

### 运行游戏

直接在浏览器中打开 `index.html` 即可运行：

```bash
# 方式 1: 直接打开
open index.html  # macOS
xdg-open index.html  # Linux
start index.html  # Windows

# 方式 2: 使用本地服务器 (推荐)
npx http-server -p 8080
# 访问 http://localhost:8080
```

## 🎮 操作说明

### 基础控制
| 按键 | 功能 |
|------|------|
| A / ← | 向左移动 |
| D / → | 向右移动 |
| W / ↑ | 跳跃 |
| J / K | 普通攻击 |
| L | 特殊技能 |

### 道具使用
| 按键 | 道具 | 效果 |
|------|------|------|
| H | 金创药 | 恢复 50% 血量 |
| M | 魔力水 | 恢复 50% 能量 |
| F | 战神符 | 30 秒无敌 |
| T | 天书 | 全屏攻击 |
| U | 太平要术 | 召唤援军 |

### 系统控制
| 按键 | 功能 |
|------|------|
| Enter / Space | 开始游戏 / 确认选择 |
| P | 暂停游戏 |
| R | 重新开始 (游戏结束时) |

## 🎯 核心特性

### 关卡系统
- ✅ **第 1 关：涿郡之战** - 教学关，对战张飞
- ✅ **第 2 关：虎牢关** - BOSS 战，对战吕布
- ✅ **第 3 关：赤壁之战** - 最终关，对战曹操
- ✅ 每关多波次敌人配置
- ✅ 关卡背景差异化

### BOSS 设计
- ✅ **张飞** - 冲锋攻击、怒吼技能
- ✅ **吕布** - 方天画戟、无双乱舞、鬼神降临
- ✅ **曹操** - 召唤士兵、剑气远程、奸雄霸业
- ✅ 三阶段血量变化
- ✅ 阶段转换特效

### 道具系统
- ✅ **金创药** - 恢复 50% 血量
- ✅ **魔力水** - 恢复 50% 能量
- ✅ **战神符** - 30 秒无敌状态
- ✅ **天书** - 全屏攻击
- ✅ **太平要术** - 召唤援军

### 连击系统
- ✅ 连击计数（2 秒内连续击中）
- ✅ 连击评价：C/B/A/S/SSS
- ✅ 连击分数倍率奖励
- ✅ 连击成就系统

### 界面系统
- ✅ 动态开始界面
- ✅ 角色选择界面（显示属性对比）
- ✅ 游戏内 HUD（血量/法力/分数/连击）
- ✅ BOSS 血条显示
- ✅ 暂停/游戏结束界面

## 🔧 技术栈

- **HTML5 Canvas** - 游戏渲染
- **ES6+** - 现代 JavaScript 语法
- **无依赖** - 纯原生实现
- **60FPS** - 流畅游戏循环
- **面向对象** - 模块化设计

## 📋 游戏流程

1. **开始界面** - 按任意键或点击"开始游戏"
2. **角色选择** - 使用 ←→ 或 A/D 选择角色，按 J 确认
3. **第 1 关** - 涿郡之战，击败 3 波敌人后对战张飞
4. **第 2 关** - 虎牢关，击败 3 波敌人后对战吕布
5. **第 3 关** - 赤壁之战，击败 3 波敌人后对战曹操
6. **通关** - 击败曹操后游戏通关

## 🎯 游戏提示

- 连击数越高，分数倍率越高（最高 3 倍）
- 合理使用道具可以扭转战局
- 每个 BOSS 有三个阶段，血量越低越强
- 注意躲避 BOSS 的蓄力技能
- 击败敌人有几率掉落道具
