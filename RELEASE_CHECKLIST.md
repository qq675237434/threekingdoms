## 🚀 发布步骤

### 1. 发布前准备

```bash
# 切换到主分支
git checkout main

# 拉取最新代码
git pull origin main

# 运行最终测试
npm test

# 启动本地服务器验证
npm run serve
```

### 2. 版本标签

```bash
# 创建版本标签
git tag -a v1.0.0 -m "Release version 1.0.0 - Initial release"

# 推送标签
git push origin v1.0.0
```

### 3. 部署

```bash
# 如果使用 Vercel
vercel --prod

# 或手动部署
# 将以下文件上传到服务器：
# - index.html
# - game.js
# - core/
# - entities/
# - ai/
# - ui/
```
