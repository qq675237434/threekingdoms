# 三国战记 - 快速测试指南

## 🎮 游戏流程

### 步骤 1: 打开游戏
访问 https://threeking.bitsclock.com

### 步骤 2: 开始界面
- 看到"三国战记"标题
- 按 **空格键** 或 **点击屏幕** 进入主菜单

### 步骤 3: 主菜单
使用 **↑/↓** 或 **W/S** 选择菜单项：
- 开始游戏 - 直接使用默认角色（关羽）开始
- 角色选择 - 选择角色（关羽/张飞/赵云）
- 游戏说明 - 查看操作说明
- 制作人员 - 制作信息

### 步骤 4: 角色选择（如果选择）
使用 **←/→** 或 **A/D** 切换角色
按 **J** 或 **Enter** 确认选择

### 步骤 5: 开始战斗！
操作：
- **A/D** 或 **←/→** - 左右移动
- **W** 或 **↑** - 跳跃
- **J** - 普通攻击
- **K** - 特殊技能
- **H** - 使用金创药
- **P** - 暂停

---

## 🐛 如果遇到问题

### 问题 1: 卡在加载界面
**解决**: 按 F12 打开控制台查看错误，刷新页面

### 问题 2: 开始界面没反应
**解决**: 确保点击了游戏画面（获取焦点），然后按空格

### 问题 3: 角色选择后没反应
**解决**: 按 J 键确认选择，等待 0.5 秒

### 问题 4: 黑屏
**解决**: 检查浏览器控制台，可能是 WebGL 不支持

## 🎯 快速开始（跳过菜单）

在控制台输入：
```javascript
window.game.startGameDirectly()
```

这会直接使用关羽开始游戏，跳过所有菜单。
