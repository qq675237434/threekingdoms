### ✅ 5. 特效系统 (effects/Effects.js)

**刀光特效 (SlashEffect)：**
- 弧形白色轨迹
- 渐变透明度
- 旋转动画
- 边缘高光

**火花特效 (SparkEffect)：**
- 10-20 个粒子
- 放射状飞散
- 重力下落
- 颜色可选

**升级光效 (LevelUpEffect)：**
- 3 层金色光环
- 扩散动画
- 脉冲效果

**屏幕震动 (ScreenShakeEffect)：**
- 随机偏移
- 强度衰减
- 持续时间可控

**伤害数字 (HitNumberEffect)：**
- 上浮动画
- 缩放效果
- 暴击标识（CRITICAL!）
- 颜色分级（普通/暴击/高伤）

**光环特效 (AuraEffect)：**
- BOSS 环绕光环
- 脉冲动画
- 颜色自定义
- 无限/定时模式

**爆炸特效 (ExplosionEffect)：**
- 中心扩散
- 碎片粒子（20 个）
- 颜色渐变（白→黄→橙→红）

**特效管理器 (EffectManager)：**
- 统一管理所有特效
- 自动清理完成特效
- 便捷创建接口

## 技术实现

### Canvas 原生绘制
- 不使用任何外部图片
- 全部使用 Canvas API 绘制
- 渐变、阴影、变换等高级效果

### 动画系统
- 帧动画（行走/攻击）
- 程序动画（摆动/脉冲）
- 物理动画（重力/粒子）

### 性能优化
- 视口外实体不渲染
- 特效自动清理
- 分层渲染

## 使用说明

1. **启动游戏**：打开 `index.html`
2. **角色选择**：按 ← → 选择角色，按攻击键确认
3. **游戏控制**：
   - 方向键：移动
   - Z/X：攻击/技能
   - P：暂停
   - R：重新开始
