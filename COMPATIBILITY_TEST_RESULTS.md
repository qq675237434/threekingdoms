## 已知问题

### 轻微问题

1. **Safari 阴影性能**
   - 问题：shadowBlur 在 Safari 上性能较低
   - 影响：BOSS 光环效果 FPS 下降 3-5
   - 解决：已降级处理，移动端禁用部分阴影

2. **Firefox 字体渲染**
   - 问题：中文字体渲染略有差异
   - 影响：UI 文字边缘不够平滑
   - 解决：使用系统字体回退

3. **移动端触控延迟**
   - 问题：部分 Android 设备触控响应延迟 ~50ms
   - 影响：操作手感略差
   - 解决：添加 touch-action: none CSS

### 不推荐的环境

| 浏览器/系统 | 版本 | 原因 |
|-------------|------|------|
| IE 11 | 所有 | 不支持 ES6、Canvas 性能差 |
| Chrome | < 60 | requestAnimationFrame 支持不完整 |
| Safari | < 11 | Canvas 性能问题 |
| Firefox | < 55 | ES6 支持不完整 |
| Android Browser | < 8 | 性能问题严重 |

## 浏览器检测结果

### 检测脚本

```javascript
window.browserSupport = {
    canvas: supportsCanvas(),
    raf: supportsRAF(),
    localStorage: supportsLocalStorage(),
    isMobile: /Android|webOS|iPhone|iPad|iPod|BlackBerry|IEMobile|Opera Mini/i.test(navigator.userAgent)
};
```

### 回退方案

1. **Canvas 不支持**
   - 显示友好提示
   - 推荐现代浏览器
   - 提供下载链接

2. **requestAnimationFrame 不支持**
   - 降级到 setTimeout
   - 限制帧率到 30 FPS

3. **LocalStorage 不支持**
   - 使用内存存储
   - 数据不持久化

## 优化建议

### 针对低端设备

1. **自动降级**
   - 检测到低性能设备时自动降低画质
   - 减少粒子数量
   - 降低阴影质量

2. **分辨率适配**
   - 移动端使用较小 Canvas
   - CSS 缩放至全屏

3. **帧率限制**
   - 低端设备限制 30 FPS
   - 减少计算压力

### 针对特定浏览器

1. **Safari**
   - 减少 shadowBlur 使用
   - 使用 CSS 动画替代部分 Canvas 动画

2. **Firefox**
   - 优化字体渲染
   - 使用系统字体

3. **移动端 Chrome**
   - 优化触控响应
   - 添加防误触机制
